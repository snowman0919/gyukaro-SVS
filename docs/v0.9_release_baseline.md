## Frozen behavior

KO, EN, and JA each render 48 kHz mono phrase audio. A +2-semitone note edit produced +200.41 cents RMVPE movement; +1-semitone PITD produced +92.52 cents. Lyric edit changed Whisper output from “하늘빛 노래” to “사랑을 담아서 내준 내 마음”. Neutral/energetic RMS was 0.078379/0.079411. Native C# `IRenderer.Render` returned non-silent 44.1 kHz editor samples.

Identity effect is real but weak: WavLM mean +0.00200 and ECAPA +0.00134 with confidence intervals crossing zero. Stable styles are neutral, breathy, and energetic. Soft, dark, and bright remain experimental relative controls. Personalized prosody is Korean-only; EN/JA use generic multilingual prosody with GYU identity/style.

## Installation requirements at v0.9

Linux aarch64 was validated with Python 3.11.14 and .NET 8.0.422. Package required an existing pinned SoulX/OmniVoice cache and compatible SoulX Python, manual environment variables, and a source-built patched OpenUtau. These are v1.0 installation gaps, not hidden v0.9 capabilities.
