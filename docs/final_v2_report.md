# Final v2 report

Hybrid v0.2 is an executable experimental phrase-level neural SVS path: full conditioning modules are connected, trained for 160 steps, decoded by a frozen pretrained codec, rendered as WAV and packaged. Stage 0 source-loop and per-note TTS/DSP vocalizers remain named baselines only. Major limits: inferred real-anchor scores, no accepted multilingual pseudo corpus, and no quality claim beyond runtime verification.
